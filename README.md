# OmniForge AI

OmniForge AI is an Unreal Engine assistant built for fast game-development iteration inside the editor. It gives you a chat-first workflow for building gameplay systems, creating and editing assets, compiling, diagnosing errors, and turning rough ideas into playable Unreal prototypes.

## What It Does

- Builds UE gameplay features from natural language prompts.
- Creates and edits C++ classes, Blueprints, UMG widgets, levels, actors, input setup, and project settings.
- Wires common Blueprint graph events and values, including lifecycle/custom event entry points, Enhanced Input actions, component overlap delegates, UMG button clicks, typed Blueprint variables, typed Blueprint functions, concrete variable get/set nodes, Dynamic Cast nodes, CreateWidget nodes, Branch/Sequence/Self flow nodes, node pin defaults, and structured pin connections.
- Runs compile and verification workflows so you can see what actually works.
- Returns structured compile evidence with exit codes, exact UBT command, log paths, detected errors, and next safe action when C++ or toolchain setup fails.
- Supports tool-driven game-development workflows for mechanics, UI, world building, assets, animation, VFX, audio, and diagnostics.
- Uses an OmniForge license key so model access, credits, routing, updates, and account limits are managed centrally.

## Best For

- Solo developers who want to prototype faster.
- Small teams building UE gameplay systems.
- Game jam workflows where speed matters.
- Designers who want AI-assisted Blueprint, UI, and level setup without manually wiring every step.
- Developers who want an assistant that can inspect, build, fix, and verify instead of only chatting.

## Install

1. Download the latest release ZIP from the Releases page.
2. Close Unreal Engine.
3. Extract the `OmniForgeAI` folder into:

   ```text
   YourProject/Plugins/Marketplace/OmniForgeAI/
   ```

   For an engine-wide install, extract it into:

   ```text
   UE_5.7/Engine/Plugins/Marketplace/OmniForgeAI/
   ```

4. Open your Unreal project.
5. Enable `OmniForge AI` from `Edit > Plugins` if it is not enabled automatically.
6. Open the OmniForge panel and paste your license key.

## Requirements

- Unreal Engine 5.7 on Windows.
- A valid OmniForge license key.
- Internet access for model, license, update, and search services.
